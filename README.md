# GameHub - Interactive Tic-Tac-Toe Game

## Objective

This project is a frontend web application built for academic purposes. It demonstrates responsive design and interactive web development through a single-page website that includes a playable Tic-Tac-Toe game.

## Technologies Used

- HTML
- CSS
- JavaScript

## Features Implemented

- Fixed navigation bar with links to Home, About, and Tic-Tac-Toe sections
- Mobile hamburger menu for smaller screens
- Active navigation link updates on scroll
- Hero section with animated background
- About section with feature cards
- Tic-Tac-Toe game with turn-based gameplay (Player X and Player O)
- Win detection across rows, columns, and diagonals
- Draw detection when all cells are filled
- Reset game functionality
- Game status display (next player, winner, or draw)
- Winning cell highlight on game win
- Responsive layout for desktop, tablet, and mobile screens
- Footer with quick links and social media links

## Steps to Run the Application

1. Open the project folder (`ModernWebsite`).
2. Open `index.html` in a web browser by clicking preview in VS code or right click and choose Open with live server.
